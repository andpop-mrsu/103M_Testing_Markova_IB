ТЕСТ-ПЛАН ДЛЯ ТРЕУГОЛЬНИКОВ

Группа 1: Валидные кейсы (Positive)
- Равносторонний: [6, 6, 6] -> "equilateral"
- Равнобедренный: [5, 5, 8] -> "isosceles"
- Разносторонний: [7, 8, 9] -> "nonequilateral"
- Float-значения: [2.5, 2.5, 4.0] -> "isosceles"

Группа 2: Ошибочные кейсы (Negative)
- Нулевая сторона: [0, 5, 5] -> Error
- Отрицательная сторона: [5, -2, 5] -> Error
- Неравенство треугольника: [1, 2, 5] -> Error
- Вырожденный случай: [2, 2, 4] -> Error
- Некорректный тип: ["10", 10, 10] -> Error
